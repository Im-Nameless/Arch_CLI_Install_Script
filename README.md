# Arch/BlackArch TUI Installer

To run the scrip;
`wget git.io/abatui && chmod +x abatui && ./abatui`

If the first link doesn't work;
`wget raw.githubusercontent.com/smyjpmu/abatui/master/abatui.sh && chmod +x abatui.sh && ./abatui.sh`
